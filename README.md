# knowledge-summary-skill

一个用于生成深度技术知识总结文档的 skill。目标是把框架、API、协议、源码模块、语言特性等主题整理成单个 Markdown 文档，让新手能看懂，老手也能复习关键实现。

## 功能

- 先输出大纲，再根据确认后的结构生成正文。
- 只使用官方文档、官方源码、标准规范等可信资料源。
- 覆盖是什么、核心概念、原理、源码、设计取舍、实操示例和边界问题。
- 强制深入到具体实现：文件路径、函数、数据结构、调用链和关键源码片段。
- 代码第一次出现时直接加入中文注释，避免重复贴两遍代码。

## 结构

```text
knowledge-summary/
├── SKILL.md
├── README.md
└── references/
    └── self-check.md
```

- `SKILL.md`：skill 的主说明文件，定义触发场景、工作流程、内容深度要求和输出规则。
- `references/self-check.md`：写完文档后的自检清单，用于检查深度、资料源、代码、结构和输出质量。
